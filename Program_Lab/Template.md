6. 다음 ``Counter`` 템플릿 클래스를 정의를 검증하는 코드를 작성하고 검증하시오. 사용할 수 있는 자료형의 제약이 있는지 검토하시오.

```cpp
template <typename T>
class Counter{
	T value;
public:
	Counter() { value = 0;}
	Counter(T n) { value = n;}
	T getValue() {return value;}
	void operator ++() {++value;}
	void operator --() {--value;}
};
```
